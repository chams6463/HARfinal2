# 예제 05: 아날로그 전압 측정 — XM ADC 입문

본 예제는 **고정 아날로그 핀(ADC_1~4)**에서 전압을 읽는 가장 기본적인 방법을 다룹니다. `XM_AnalogReadMillivolts()`로 밀리볼트(mV) 단위의 직관적인 전압 값을 바로 읽고, 임계치 기반으로 LED를 제어합니다.

## 🎯 학습 목표 (Objective)

* `XM_AnalogReadMillivolts()`로 아날로그 핀의 전압을 **밀리볼트(mV)** 단위로 읽는 방법을 학습합니다.
* ADC raw 값을 수동 변환할 필요 없이 API 하나로 전압을 얻을 수 있음을 확인합니다.
* 임계치(Threshold) 기반 판단 로직을 구현합니다.

## ⚙️ 동작 원리 (How it Works)

* **전압 읽기:** `XM_AnalogReadMillivolts(XM_EXT_ADC_1)`은 PA0 핀에 연결된 센서의 전압을 0~3300 mV 범위로 반환합니다.
* **내부 동작:** 네이티브 ADC raw 값에서 직접 밀리볼트로 변환하므로, `XM_SetAnalogReadResolution()` 설정에 영향받지 않습니다.
* **임계치 판단:** 전압이 2000mV(2.0V)를 초과하면 내부 LED를 켜서 경고합니다.

## 🔌 하드웨어 연결

```
3.3V ─── 가변저항(VR) ──┬── XM_EXT_ADC_1 (PA0)
                        └── GND
```

## 🚀 실행 방법 (How to Use)

1. **하드웨어 연결:** 가변저항 또는 아날로그 센서의 출력을 `XM_EXT_ADC_1`에 연결합니다.
2. 코드를 업로드하고 전원을 켭니다.
3. **테스트:** 가변저항을 돌려 전압을 변화시키면, 2.0V 초과 시 내장 LED가 켜지는 것을 확인합니다.

## 📚 ADC 튜토리얼 시리즈

이 예제는 XM ADC 튜토리얼 시리즈의 **첫 번째**입니다:

| 예제 | 난이도 | 주제 |
|------|--------|------|
| **05 (본 예제)** | 초급 | 고정 ADC 핀 전압 읽기 |
| 05a | 초급 | DIO→ADC 전환으로 FSR 센서 읽기 |
| 05b | 중급 | FSR 8채널 일괄 전환 + Resolution 설정 |
| 05c | 중급 | 고정 ADC + DIO→ADC 혼합 사용 (12핀 전체) |
| 05d | 응용 | DIO/ADC 혼합 모드 — 버튼 + LED + FSR 실전 |

## 💡 직접 해보기 (Things to Try)

* **임계치 변경:** `2000`을 `1000`이나 `2500`으로 바꿔 반응 지점을 조절해보세요.
* **raw 값 비교:** `XM_AnalogRead(XM_EXT_ADC_1)`과 `XM_AnalogReadMillivolts()`의 차이를 비교해보세요.
* 다음 예제 **05a**에서 DIO 핀을 ADC로 전환하여 FSR 센서를 읽어봅니다.
