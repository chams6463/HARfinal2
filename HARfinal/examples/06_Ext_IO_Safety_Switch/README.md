# 예제 06: 외부 안전 제어 (External IO Safety)

본 예제는 로봇 시스템에서 가장 중요한 **안전(Safety)** 기능을 구현합니다. 외부 리미트 스위치(Limit Switch)나 비상 정지 버튼(E-Stop)과 같은 하드웨어 신호를 감지하여, 로봇을 즉시 정지시키고 **에러(Error)** 상태로 전환하는 실전 로직을 다룹니다.

## 🎯 학습 목표 (Objective)

* 외부 센서(리미트 스위치) 입력을 시스템의 **인터럽트(또는 트리거)** 조건으로 사용하는 법을 배웁니다.
* TSM(Task State Machine)을 활용하여 정상 동작(`ACTIVE`)과 비상 정지(`ERROR`) 상태를 명확히 분리합니다.
* 에러 발생 시 모터 출력을 차단하고, 사용자의 확인(Reset) 과정을 거쳐 복구하는 안전 시퀀스를 구현합니다.

## ⚙️ 동작 원리 (How it Works)

이 시스템은 3가지 상태를 순환하며 동작합니다.

1.  **STANDBY (대기):** 시작 버튼(`XM_EXT_DIO_3`) 입력을 기다립니다.
2.  **ACTIVE (동작):** 토크 제어를 수행하며 모터를 구동합니다. 동시에 안전 센서(`XM_EXT_DIO_4`)를 감시합니다.
    * 만약 안전 센서가 눌리면(`HIGH`), 즉시 `ERROR` 상태로 전이합니다.
3.  **ERROR (비상 정지):**
    * 진입 즉시 `XM_SetControlMode(XM_CTRL_MONITOR)`를 호출하여 **모터 출력을 0으로 차단**합니다.
    * 빨간색 LED를 깜빡여 위험을 알립니다.
    * 사용자가 내부 버튼(`XM_BTN_1`)을 눌러 에러를 해제할 때까지 대기합니다.

## 🚀 실행 방법 (How to Use)

1.  **하드웨어 연결:**
    * **시작 버튼:** `XM_EXT_DIO_3`와 `GND` 사이에 연결 (Input Pullup 사용).
    * **리미트 스위치:** `XM_EXT_DIO_4`와 `3.3V` 사이에 연결, `GND`와 풀다운 저항 연결 (또는 3.3V와 직접 연결하여 테스트 시 전선 접촉).
2.  코드를 빌드하고 업로드합니다.
3.  시작 버튼(Pin 3)을 눌러 `XM_STATE_ACTIVE` 모드(LED 2 켜짐)로 진입합니다.
4.  동작 중 리미트 스위치(Pin 4)를 누릅니다(3.3V 인가).
    * LED 2가 꺼지고 LED 1(빨강)이 빠르게 깜빡이며 시스템이 정지하는 것을 확인합니다.
5.  **내부 버튼 1**을 눌러 에러를 해제하고 `XM_STATE_STANDBY`로 복귀합니다.

## 💡 직접 해보기 (Things to Try)

* **이중 안전장치:** 리미트 스위치뿐만 아니라, 모터의 각도가 특정 범위(예: 90도)를 넘어가면 소프트웨어적으로 에러를 발생시키는 코드를 `Active_Loop`에 추가해보세요.
* **부저(Buzzer) 추가:** 에러 상태일 때 또 다른 출력 핀을 제어하여 경고음을 내보세요.
