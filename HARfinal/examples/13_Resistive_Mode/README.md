# 예제 13: 저항 모드 (Resistive Mode)

본 예제는 H10의 내장 기능을 활용하여, 물속에서 걷는 듯한 저항감(운동 효과)을 주는 애플리케이션입니다. 복잡한 토크 제어 없이 파라미터 설정만으로 고급 제어 모드를 구현합니다.

## 🎯 학습 목표 (Objective)

* H10의 **내장 제어 모드**를 활용하는 방법을 이해합니다.
* 이벤트 기반 업데이트(Event-driven Update)의 개념을 익힙니다. (값이 바뀔 때만 통신)
* `XM_SetResistiveGain` API를 사용하여 로봇의 물리적 특성(저항성)을 실시간으로 변경합니다.

## ⚙️ 동작 원리 (How it Works)

이 예제는 XM10이 직접 2ms마다 토크를 계산해서 쏘는 것이 아닙니다. 대신 H10에게 "저항 모드로 동작하고, 강도는 X로 해"라고 설정만 해줍니다.

1.  **입력:** 사용자가 H10의 보조 레벨 버튼(+/-)을 누릅니다.
2.  **감지:** XM10은 `XM.status.h10.h10AssistLevel` 값이 변한 것을 감지합니다.
3.  **계산:** 레벨(0~10)에 비례하여 저항 이득(Gain) 값을 계산합니다. (레벨이 높을수록 더 뻑뻑하게)
4.  **전송:** `XM_SetResistiveGain`을 호출하여 변경된 게인 값을 H10으로 보냅니다. 통신 부하를 줄이기 위해 **값이 변했을 때만** 전송합니다.

## 🚀 실행 방법 (How to Use)

1.  H10 슈트와 XM10을 연결하고 전원을 켭니다.
2.  코드를 실행하고 **버튼 2**를 눌러 `XM_STATE_ACTIVE` 모드로 진입합니다.
3.  슈트를 입고 다리를 움직여봅니다. (저항감이 느껴집니다)
4.  슈트 본체의 **+ 버튼**을 눌러 레벨을 올립니다. 다리가 더 무거워지는지(운동 강도 증가) 확인합니다.
5.  **- 버튼**을 눌러 레벨을 내리면 다시 가벼워지는지 확인합니다.

## 💡 직접 해보기 (Things to Try)

* **게인 튜닝:** 코드 상단의 `RESISTIVE_MODE_MAX_GAIN` 값을 변경하여 저항감의 최대치를 조절해보세요.
* **비대칭 설정:** 왼쪽 다리와 오른쪽 다리의 게인을 다르게 설정하여, 한쪽 다리만 재활 훈련을 하는 시나리오를 구현해보세요.
